# Dữ liệu
Nguồn: [Traffic Flow data in Ho Chi Minh City, Viet Nam (Kaggle)](https://www.kaggle.com/datasets/thanhnguyen2612/traffic-flow-data-in-ho-chi-minh-city-viet-nam)

Tập dữ liệu bao gồm 5 file:
- `nodes.csv`: Chứa danh sách các cảm biến và tọa độ. Gồm 3 cột, bao gồm id của sensor, kinh độ và vĩ độ của cảm biến.
- `segments.csv`: Chứa thông tin về các đường trên TP. HCM. Gồm 5 cột: Id của đường, cấp công trình giao thông, tốc độ tối đa cho phép, tên đường, loại đường
